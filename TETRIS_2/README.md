Run the app using [http-server](https://www.npmjs.com/package/http-server).

Controls:
```
Escape: pause or continue (also after game over)
ArrowDown: speed up falling
ArrowLeft: move piece left
ArrowRight: move piece right
Space: rotate piece clockwise by 90 degrees
```

Modified from: [here](https://github.com/yasinishyn/tetris_js).
